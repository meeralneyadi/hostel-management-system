# Hostel Management System (HMS)

A web-based Hostel Management System developed using Java, JSP, Servlets, JDBC, and MySQL.

## Overview

The Hostel Management System (HMS) is designed to simplify hostel operations by providing separate interfaces for Administrators, Staff Members, and Residents.

The system supports room allocation, booking requests, support requests, room reassignment, checkout requests, and multi-building hostel management.

## Features

### Administrator

* Manage rooms
* Manage residents
* Assign rooms to buildings
* Approve and reject booking requests
* Approve and reject checkout requests
* Change resident room assignments
* Monitor support requests

### Staff

* View support requests
* Update request status
* Track maintenance workflow

### Resident

* View available rooms
* Submit booking requests
* Track booking status
* Submit support requests
* View support request history
* Submit checkout requests
* Track checkout status
